Consider the file Census 1000. This file contains the most common surnames found in USA.

TASK 1: Read the table, show its methods. 

$table = import-csv D:\PowerShellApplications\Census1000.csv
$table | Get-Member


TASK 2: Find the most popular surname among hispanics and the rate of it:

$table = import-csv D:\PowerShellApplications\Census1000.csv
$table | sort -Property pcthispanic -descending | format-table -property name, pcthispanic

Bu noktada dikkat etmek gereken şey; format-table metodunu kullandıktan sonra indexleme yapamayız!! Format-table sadece görüntülemek içindir.

Bir yöntem:

$table = import-csv D:\PowerShellApplications\Census1000.csv
($table | sort -Property pcthispanic -descending)[0].name
($table | sort -Property pcthispanic -descending)[0].pcthispanic

Başka bir yöntem:

$table = import-csv D:\PowerShellApplications\Census1000.csv
$table | sort -Property pcthispanic -descending | select -first 1
